# AirQualityMonitor 使用指南

## 系统功能

ESP32 空气质量监测仪，实时监测以下指标并通过 OLED 屏和蓝牙（BLE）输出：

| 指标 | 传感器 | 单位 |
|------|--------|------|
| PM1.0 / PM2.5 / PM10 | PMS7003 | μg/m³ |
| 温度 / 湿度 | 21VOC 模块 | ℃ / %RH |
| VOC（挥发性有机物） | 21VOC 模块 | μg/m³ |
| HCHO（甲醛） | 21VOC 模块 | μg/m³ |
| eCO₂（等效二氧化碳） | 21VOC 模块 | ppm |
| 光照强度 | BH1750 | lux |

---

## 蓝牙连接与数据

1. 上电后设备广播 **AirQualityMonitor**
2. 使用支持 NUS 的 App（如 Serial Bluetooth Terminal）连接
3. 设备 **每秒自动推送** 以下格式数据：

```
PM2.5:25 PM10:35 T:26.5 H:60.5 VOC:120 HCHO:30 eCO2:450 LUX:320 Grade:1 Alarm:ON
```

### 字段说明

| 字段 | 说明 | 示例值 | 等级码含义 |
|------|------|--------|-----------|
| PM2.5 / PM10 | 颗粒物浓度 (μg/m³) | `25` | — |
| T | 温度 (℃) | `26.5` | — |
| H | 湿度 (%RH) | `60.5` | — |
| VOC | 挥发性有机物 (μg/m³) | `120` | — |
| HCHO | 甲醛 (μg/m³) | `30` | — |
| eCO2 | 等效 CO₂ (ppm) | `450` | — |
| LUX | 光照强度 (lux) | `320` | — |
| Grade | 空气质量等级 | `1` | 0=Good · 1=Fair · 2=LightPol · 3=ModPol · 4=HeavyPol |
| Alarm | 蜂鸣器状态 | `ON` | — |

> Grade 根据 PM2.5 浓度划分：≤35 / 75 / 115 / 150 μg/m³。

### 手机指令

| 指令 | 功能 |
|------|------|
| `'1'` | 开启蜂鸣器 |
| `'0'` | 静音 10 分钟 |
| `'R'` | 立刻刷新一次数据推送 |

---

## OLED 屏幕布局

```
AQ:Good   Sunny         ← 等级 + 天气
P1:10 P2:25 10:35       ← PM1.0 / PM2.5 / PM10
VOC:120 HCHO:30         ← VOC + 甲醛
eCO2:450ppm Lux:320     ← CO₂ + 光照
T:26.5 H:60.5%          ← 温湿度
```

- **ALARM**（反白）— 蜂鸣器正在响
- **MUTED** — 蜂鸣器已被静音

---

## 蜂鸣器报警

- **触发条件**：PM2.5 > 75 或 VOC > 500
- **声音模式**：先短促 200ms → 然后 500ms 响 / 500ms 停 循环
- **静音方式**：长按板载 **BOOT 按钮** 1 秒，或手机发送 `'0'`
- **自动恢复**：静音后 10 分钟自动恢复报警

---

## 构建与部署

```bash
pio run -e esp32dev                # 编译
pio run -e esp32dev -t upload      # 上传至 COM5
pio device monitor -p COM5 -b 115200   # 查看串口日志
```

上电后串口输出 `Bluetooth Ready!` 即表示启动成功。
