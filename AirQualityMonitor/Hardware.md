### UART 分配

| UART | TX 引脚 | RX 引脚 | 外设 | 波特率 |
|------|---------|---------|------|--------|
| UART0 | GPIO1 | GPIO3 | USB-UART（调试） | 115200 |
| UART1 | GPIO19 | GPIO18 | 21VOC 传感器 | 9600 |
| UART2 | GPIO17 | GPIO16 | PMS7003 | 9600 |

> 两个传感器上电后均主动持续上报数据。ESP32 的 TX 引脚（GPIO17、GPIO19）已连接以备后续指令下发，当前固件未主动使用。

### I2C 总线

共享 I2C 总线，100kHz 标准模式：

| 设备 | 地址 | 备注 |
|------|------|------|
| SSD1306 OLED | 0x3C | 128×64 像素 |
| BH1750 | 0x23 | ADDR 引脚 = LOW |

- SDA/SCL 各 4.7kΩ 上拉到 3.3V
- 如模块自带板上上拉，可省略 R2/R3

## 3. 各子系统设计

### 电源

```
USB 5V ──┬── LDO AMS1117-3.3 ──┬── VCC_3V3
          │   IN         OUT    │
        [C1]  [C3]    [C2]   [C4]
        10μF 100nF  10μF   100nF
          │     │      │      │
         GND   GND   GND    GND
```

- 3.3V 峰值电流约 560mA（ESP32 BLE 发射 ~500mA + OLED 20mA + 其它），AMS1117 额定 1A 裕量充足
- 5V 轨：PMS7003 (~150mA) + 21VOC (~50mA) 直接从 USB 取电
- C3/C4 尽量靠近 ESP32 3.3V/GND 引脚（<5mm）
- 可选：USB 5V 输入端串 SS14 做防反接，加 500mA PTC 做过流保护

### OLED 显示屏 (SSD1306)

```
VCC_3V3 ── VCC
GND    ── GND
GPIO21 ── SDA
GPIO22 ── SCL
```

- 标准 0.96 寸 I2C OLED 模块，4 引脚
- 无需外部复位（固件中 `OLED_RESET = -1`）
- 典型功耗：15-25mA @ 3.3V

### PMS7003 颗粒物传感器

```
+5V0  ── VCC（第 1 脚）
GND   ── GND（第 2 脚）
GPIO16 ── TX（第 4 脚，传感器数据）
GPIO17 ── RX（第 3 脚，预留）
        ── SET（第 5 脚，悬空 = 主动模式）
        ── RES（第 6 脚，悬空 = 正常工作）
```

- 连接器：1.27mm 间距 2×5 母座（JST SH 兼容）
- 校验方式：前 30 字节累加和等于校验和字段（大端 uint16）

**数据帧（32 字节，大端）：**

| 偏移 | 字段 | 说明 |
|------|------|------|
| 0-1 | 帧头 | 0x42, 0x4D |
| 2-3 | 帧长度 | 0x001C (28) |
| 4-9 | PM1.0/2.5/10 CF=1 | 标准颗粒物 (μg/m³) |
| 10-15 | PM1.0/2.5/10 大气 | 大气环境 (μg/m³) |
| 16-27 | 六档颗粒计数 | ≥0.3/0.5/1.0/2.5/5.0/10μm 个/0.1L |
| 28 | 版本号 | — |
| 29 | 错误码 | 0 = 正常 |
| 30-31 | 校验和 | 前 30 字节累加和 |

### 21VOC 五合一传感器

```
+5V0  ── VCC
GND   ── GND
GPIO18 ── TX（传感器数据）
GPIO19 ── RX（预留）
```

- 上电后约每 1 秒主动上报，9600 波特率

**数据帧（12 字节，大端）：**

| 偏移 | 字段 | 说明 |
|------|------|------|
| 0 | 帧头 | 0x2C |
| 1-2 | VOC | μg/m³ |
| 3-4 | HCHO 甲醛 | μg/m³ |
| 5-6 | eCO₂ | ppm |
| 7-8 | 温度 | 有符号补码，÷10 = ℃ |
| 9-10 | 湿度 | ÷10 = %RH |
| 11 | 校验和 | 前 11 字节累加取补码，使 12 字节和 mod 256 = 0 |

### BH1750 光照传感器

```
VCC_3V3 ── VCC
GND    ── GND
GPIO21 ── SDA
GPIO22 ── SCL
        ADDR ── GND（地址 0x23）
```

- 连续高分辨率模式（指令 0x10），测量时间 120-180ms
- 原始值 ÷ 1.2 = lux（整数运算 `raw*10/12`）
- 传感器需放置在能无遮挡接收环境光的位置

### 蜂鸣器驱动

```
+5V0 ── 蜂鸣器(+) ── 蜂鸣器(-) ── 集电极(Q1 S8050)
                                    发射极 ── GND
                                    基极 ──[1kΩ]── GPIO25
```

- NPN 三极管（S8050）共射极接法。GPIO25 高电平通过 1kΩ 基极电阻驱动约 3.3mA 基极电流，使三极管饱和导通
- 蜂鸣器从 5V 取电，不走 GPIO，减轻 3.3V LDO 负担
- 如使用线圈式机械蜂鸣器，需并联续流二极管（1N4148，阴极接 5V）

### 静音按键

使用核心板 BOOT 按钮（GPIO 0），PCB 上 SW1 位置不焊。

- GPIO 0 板载 10kΩ 上拉至 3.3V，空闲 HIGH，按下 = LOW
- ⚠️ Strapping 引脚：上电时若按住则进入下载模式
- 固件长按 1 秒才触发，防止误触

### 编程接口

```
J2：1×6 弯针排针 (2.54mm)
1: EN / 2: IO0 / 3: TXD / 4: RXD / 5: 3.3V / 6: GND
```

- USB-UART 桥接推荐 CH340C（内置自动复位时序），可选 CP2102、FT232RL

## 4. 功耗预算

| 器件 | 电压 | 典型电流 | 最大电流 |
|------|------|----------|----------|
| ESP32-WROOM-32E | 3.3V | 130mA | 500mA |
| OLED SSD1306 | 3.3V | 15mA | 25mA |
| BH1750 | 3.3V | 0.12mA | 0.2mA |
| 蜂鸣器（鸣响中） | 5V | 40mA | 80mA |
| PMS7003 | 5V | 100mA | 150mA |
| 21VOC 传感器 | 5V | 30mA | 50mA |
| **合计** | | **~315mA** | **~805mA** |

**电源选型建议：** USB 5V / 2A 适配器，LDO AMS1117-3.3（1A）。

> ESP32 BLE 发射瞬间电流可达 ~500mA（微秒级），须在 VDD33 引脚旁放 100nF + 10μF 去耦电容，走线尽量短。

## 5. PCB Layout 建议

| 项目 | 要求 |
|------|------|
| 板层 | 双面板 FR4，1.6mm，1oz 铜 |
| 布局优先级 | ESP32 模组 > 传感器连接器 > 电源 > OLED |

- **电源走线：** 3.3V 采用星形拓扑，不走菊花链；1oz 铜厚下 0.5mm 可载 1A；底层整板铺 GND 铜皮
- **I2C 总线：** SDA/SCL 走线 <100mm，上拉电阻靠近 ESP32，平行走线中间夹 GND
- **UART 传感器：** 9600 波特率低速信号，走线长度不敏感，远离 BLE 天线区域
- **BLE 天线：** 模组顶端所有层禁铺铜，天线投影区域约 15mm 内无金属器件/走线
- **传感器布局：** PMS7003 靠板边保证空气流通；BH1750 留环境光通路；21VOC 远离热源；LDO 远离 ESP32 放置
- **散热：** AMS1117 输出焊盘做约 100mm² 铜皮辅助散热

## 7. 上电验证清单

1. 焊接前万用表确认 +5V / +3.3V 正确
2. 测量 5V-GND / 3.3V-GND 电阻 >1kΩ（无短路）
3. USB 供电，3.3V = 3.30V ± 0.15V
4. 通过 J2 烧录 LED 闪烁测试程序
5. 串口 115200 可看到启动信息
6. I2C 扫描检测到 0x23 (BH1750) 和 0x3C (OLED)
7. 逐个连接 PMS7003 / 21VOC，检查串口数据帧
8. nRF Connect 确认 BLE 广播名 "AirQualityMonitor"
9. 烧录正式固件全功能测试
